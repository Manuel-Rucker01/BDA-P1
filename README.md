# BDA-P1
This is the repository for the first BDA task

To use, first install the python packages required
pip install -r requirements.txt

Then, install Java (JDK 17) and set the environment variable.
Link to install: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html

windows: [System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Java\jdk-17", "User")
